How to Play GUILTY

🎯 Goal

Find the culprit by analyzing suspect traits and using logical deduction.

🔍 How It Works

  1. Read the Crime: Each day features a new mystery with detailed clues.
  2. Study the Initial Suspect: This person is NOT guilty, but their trait colors show how they compare to the real culprit.
  3. Analyze Feedback:
    Green: This trait exactly matches the culprit
    Yellow: This trait is close (±1 suspicion level)
    Gray: This trait is far from the culprit
  4. Make Guesses: Click "Accuse" button to guess who the culprit is. Each trait will show colored feedback.
  5. Use Logic: If "Museum Access: Staff" shows green, the culprit has Staff access!

💡 Tips

  • Use "Accuse" button to guess who the culprit is
  • Use "Exonerate" button to mark suspects as innocent (helps narrow down choices)
  • Exonerated suspects can be restored with the "Restore" button
  • Check the Trait Guide to see suspicion levels for each trait value
  • In Easy mode, yellow traits show arrows (↑↓) indicating if you need higher or lower suspicion
  • Focus on green traits first - they're exact matches!
  • Build a daily streak by solving each day's mystery

🏆 Difficulty Levels

  • Easy: 8 guesses, helpful arrows on close traits
  • Medium: 6 guesses, standard feedback
  • Hard: 4 guesses, prove your deduction skills!
